Affichage

Comment afficher un message à l'écran ?

Choisit la bonne réponse.





Solution : 01.

La question n'est pas de savoir si Pokémon ou Dragon Ball sont bons ou mauvais mais d'utiliser la bonne syntaxe pour afficher un message.

La bonne syntaxe est : print("le message à afficher"), le mot clef print puis des () où on indique dedans ce qu'on veut afficher.

Il faut mettre entre " " (guillemets) le texte à afficher sinon cela le fonctionne pas.
